This page contains links to example files that to help you get started using KDSmart and KDXplore.

You can use these files as templates by replacing them with your data (trials, traits or tags) and then importing them into KDSmart or KDXplore.

The example files belong to three categories:

- Trials that contain plot-level data
- Trials that contain subplot-level data (as well as plot-level)
- Additional template files for traits and tags

.. admonition:: Definition: Plots & Subplots

     Trials contain multiple *plots* - areas, spaces or even pots, uniquely identified by a plot ID or by coordinate pairs (Column/row ID), or both. *Subplots* are individual specimens within a plot.

|br|

Subplot-level Example Files
===========================

|br|



Trial With Subplot Data For a Particular Subplot
---------------------------------------------------

The first three plots in this example file have values for the trait *Plant_Height* for subplot 2.


**Download link:** :download:`Download <http://www.kddart.org/public/KDSXsample6_Specimen_Value_particular_subplot.csv>`

|br|



Trial with Subplot Data For a Particular Subplot and Trait Instance
----------------------------------------------------------------------

This example file contains the first three plots with values for:

- The trait *Plant_Height:* Instances 1 and 2; and
- Subplots 1 and 2.


**Download link:** :download:`Download <http://www.kddart.org/public/KDSXsample7_Specimen_value_particular_subplot_and_instance.csv>`

|br|



Trial with Subplot Data For All Subplots
-------------------------------------------

The trial contains subplots specified by subplot count and contains data for every subplot.

This example file contains the first three plots with values for:

- The trait *Plant_Height:* Instances 1 and 2; and
- All subplots.


**Download link:** :download:`Download <http://www.kddart.org/public/KDSXsample8_Specimen_Count_and_data_for_every_subplot.csv>`


|br|

Trial Containing Subplot Tags
-------------------------------

This example trial contains tags on the subplot level.

- New tags created with this trial import will not have a description. If this is an issue, import (i.e. create) the tags by using a tag definition file (see above `Tags File`_).
- This trial has no traits.


**Download link:** :download:`Download <http://www.kddart.org/public/KDSXsample9_wheat-with-data.csv>`

|br|


Additional Example Files
=========================

|br|

Traits File
-----------

Traits files contain `trait <http://www.kddart.org/help/kdsmart/html/traits.html#traits>`_ attributes but no trial data.


The example file below is of traits that could be found in a wheat trial.

**Download link:** :download:`Download <http://www.kddart.org/public/KDSXsample3_wheat-traits.csv>`

|br|

Tags File
---------

Tags files contain `tag <http://www.kddart.org/help/kdsmart/html/tags.html>`_ attributes but no trial data.

**Download link:** :download:`Download <http://www.kddart.org/public/KDSXsample2_tags.csv>`

An additional tags example is provided above in the `Trial Containing Subplot Tags`_, containing subplot data tags.

|br|
